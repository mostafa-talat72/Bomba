## الألوان المستخدمة

### الطاولات المحجوزة
- الخلفية: `from-red-50 to-red-100` (فاتح) / `from-red-900/30 to-red-800/20` (داكن)
- الحدود: `border-red-400` (فاتح) / `border-red-700` (داكن)
- الشارة: `bg-red-600` مع `animate-pulse`

### الطاولات الفارغة
- الخلفية: `from-green-50 to-green-100` (فاتح) / `from-green-900/30 to-green-800/20` (داكن)
- الحدود: `border-green-300` (فاتح) / `border-green-700` (داكن)
- الشارة: `bg-green-600`

### الفواتير غير المدفوعة
- الخلفية: `from-orange-50 to-red-50` (فاتح) / `from-orange-900/20 to-red-900/20` (داكن)
- الحدود: `border-orange-300` (فاتح) / `border-orange-700` (داكن)

### الفواتير المدفوعة
- الخلفية: `from-green-50 to-emerald-50` (فاتح) / `from-green-900/20 to-emerald-900/20` (داكن)
- الحدود: `border-green-300` (فاتح) / `border-green-700` (داكن)

## التأثيرات التفاعلية

### عند التمرير (Hover)
- تكبير طفيف: `hover:scale-105` أو `hover:scale-[1.02]`
- ظلال ديناميكية: `hover:shadow-lg` أو `hover:shadow-xl`
- ظلال ملونة: `hover:shadow-orange-200` حسب اللون
- تغيير لون الحدود: `hover:border-orange-400`

### الانتقالات (Transitions)
- مدة الانتقال: `duration-200` أو `duration-300`
- نوع الانتقال: `transition-all` للتأثيرات الشاملة
- تحويلات: `transform` للتكبير والحركة

## الميزات الإضافية

### دعم الوضع الداكن
- جميع التحسينات تدعم الوضع الداكن بالكامل
- ألوان متناسقة في كلا الوضعين
- تباين واضح للقراءة السهلة

### الاستجابة (Responsive)
- التصميم يعمل على جميع أحجام الشاشات
- استخدام Grid responsive للبطاقات
- أحجام خطوط متجاوبة
- كروت الطاولات تتكيف مع الشاشات الصغيرة

### الأداء
- استخدام `memo` للمكونات المتكررة
- أنيميشن CSS بدلاً من JavaScript
- تحسينات GPU acceleration
- تأثيرات Hover سلسة بدون تأخير

### تفاصيل كروت الطاولات الجديدة
- **الشارة الرئيسية**: دائرة كبيرة (64x64px) في الزاوية العلوية اليمنى
- **أيقونة الطاولة**: داخل مربع ملون (64x64px) بتدرج لوني
- **عداد الفواتير**: شارة صغيرة (32x32px) في الزاوية السفلية اليسرى
- **رقم الطاولة**: خط كبير (2xl) وواضح
- **نص "طاولة"**: تحت الرقم بخط صغير
- **تأثير Hover**: تكبير 110% + رفع للأعلى + ظل ضخم + طبقة شفافة ملونة
