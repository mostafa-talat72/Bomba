## 🔧 كيفية الاستخدام

### إضافة ترجمة جديدة لأي لغة
```bash
# استخدم الأداة المساعدة
node src/i18n/create-translation-template.js es

# أو انسخ en.json يدوياً
cp src/i18n/locales/en.json src/i18n/locales/es.json

# ثم ترجم المحتوى
```

### استخدام الترجمة في المكونات
```typescript
import { useTranslation } from 'react-i18next';

function MyComponent() {
  const { t } = useTranslation();
  
  return (
    <div>
      <h1>{t('common.welcome')}</h1>
      <button>{t('common.save')}</button>
    </div>
  );
}
```

### الحصول على معلومات اللغة
```typescript
import { WORLD_LANGUAGES, getLanguageByCode, isLanguageRTL } from '../../shared/languages';

// الحصول على معلومات لغة معينة
const arabic = getLanguageByCode('ar');
// { code: 'ar', name: 'Arabic', nativeName: 'العربية', rtl: true }

// التحقق من اتجاه اللغة
const isRTL = isLanguageRTL('ar'); // true
```
