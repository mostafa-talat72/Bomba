# Bomba - نظام إدارة شامل للمقاهي وصالات الألعاب

نظام متكامل لإدارة المقاهي والمطاعم وصالات الألعاب (بلايستيشن + كمبيوتر) مع واجهة عربية كاملة، يدعم الفوترة، المخزون، التقارير، الرواتب، والأجهزة.

---

## ✨ المميزات الرئيسية

### 🎮 إدارة صالات الألعاب
- **إدارة أجهزة البلايستيشن**: تشغيل/إيقاف الجلسات، تتبع الوقت والتكلفة
- **إدارة أجهزة الكمبيوتر**: جلسات بساعة مفتوحة، تسعير مرن
- **تغيير عدد الدراعات** أثناء الجلسة مع حساب التكلفة الدقيق
- **أسعار متغيرة**: سعر مختلف حسب عدد الدراعات (1-4)
- **عرض التكلفة التفصيلية**: تكلفة كل فترة زمنية في الجلسة

### ☕ إدارة الكافيه والمطعم
- **إضافة الطلبات**: للطاولات مع اختيار الأصناف والكميات
- **إدارة قائمة الطعام**: أصناف، فئات، أقسام، أسعار، صور
- **إدارة الطاولات**: تنظيم في أقسام، متابعة الحالة (فارغة/مشغولة/محجوزة)
- **شاشة المطبخ**: عرض الطلبات في الوقت الفعلي (قيد التحضير/جاهز)
- **تتبع وقت التحضير**: لكل صنف مع تحضير جزئي

### 💰 الفوترة والدفع
- **نظام فوترة متكامل**: إنشاء وإدارة الفواتير
- **الدفع الجزئي**: دفع أصناف محددة فقط من الفاتورة
- **طرق دفع متعددة**: كاش، كارد، تحويل بنكي، دفع مختلط
- **خصم على الفاتورة**: خصم مباشر أو بنسبة مئوية
- **رمز QR تلقائي**: لكل فاتورة لمشاركتها مع العميل
- **تتبع حالة الفاتورة**: مسودة، مدفوع جزئيًا، مدفوع بالكامل، ملغي، متأخر

### 📦 المخزون والمستودع
- **إدارة المخزون**: تتبع الكميات، حد أدنى/أقصى، حركات المخزون
- **المستودع**: مخزون كبير مع إمكانية التحويل للمخزون النشط
- **المواد الخام**: ربط الأصناف بالمكونات، التحقق من توفر المخزون
- **تتبع حركة المخزون**: سجل كامل بجميع التغييرات

### 📊 التقارير والتحليلات
- **لوحة التحكم**: إحصائيات فورية (إيرادات اليوم، الجلسات النشطة، الطلبات المعلقة)
- **تقارير المبيعات**: يومي، أسبوعي، شهري، سنوي + فلتر مخصص
- **تقارير الجلسات**: بلايستيشن وكمبيوتر مع توزيع حسب الأجهزة والدراعات
- **التقرير المالي**: إيرادات، تكاليف، أرباح، هامش ربح
- **تقرير الأصناف المباعة**: عرض تفصيلي لكل صنف مع فلتر وبحث
- **تقرير الاستهلاك**: مقارنة المبيعات مع استهلاك المخزون
- **تحليل ساعات الذروة**: أكثر الأوقات ازدحامًا
- **أداء الموظفين**: عدد الطلبات والجلسات والإيرادات لكل موظف
- **تصدير**: Excel و PDF لجميع التقارير

### 👥 إدارة المستخدمين والصلاحيات
- **صلاحيات دقيقة**: 30+ صلاحية للتحكم في الوصول
- **أدوار**: مدير، موظف، كاشير، مطبخ، مالك
- **إدارة الموظفين**: بيانات شخصية، صلاحيات، أقسام

### 💳 الرواتب والحضور
- **إدارة الرواتب**: صرف رواتب، خصومات، مكافآت، سلف
- **تسجيل الحضور**: حضور وانصراف الموظفين
- **سندات الصرف**: إدارة السلف والمكافآت والخصومات

### 💸 إدارة التكاليف
- **تسجيل المصروفات**: فئات متعددة
- **تقارير التكاليف**: تحليل المصروفات حسب الفئة والفترة

### 🌐 الدعم الدولي
- **أكثر من 130 لغة**: ترجمة كاملة للواجهة
- **دعم RTL**: عرض من اليمين لليسار للعربية والفارسية والأردية

### 🔔 الإشعارات
- **إشعارات فورية**: عبر Socket.IO
- **أنواع**: نظام، جلسة، طلب، مخزون، فاتورة
- **مستويات أولوية**: منخفض، متوسط، عالي، عاجل

### 🔄 المزامنة
- **قاعدة بيانات مزدوجة**: MongoDB محلي + MongoDB Atlas (سحابي)
- **مزامنة تلقائية**: في الاتجاهين مع إدارة التعارضات

## 🔧 التثبيت والتشغيل

```bash
# 1. استنساخ المشروع
git clone <repository-url>
cd bomba-management-system

# 2. تثبيت جميع المكتبات (فرونت + باك)
npm run install:all

# 3. إعداد متغيرات البيئة
# انسخ ملف .env.example إلى .env في مجلد server
cd server
cp .env.example .env
# عدّل الإعدادات (قاعدة البيانات، البريد الإلكتروني، إلخ)
cd ..

# 4. تشغيل وضع التطوير (السيرفر + الواجهة معًا)
npm run dev

# 5. بناء المشروع للإنتاج
npm run build:all
```

### متغيرات البيئة (server/.env)

```env
# قاعدة البيانات المحلية
MONGO_URI=mongodb://localhost:27017/bomba?replicaSet=rs0

# قاعدة بيانات Atlas (اختياري للمزامنة)
MONGODB_ATLAS_URI=mongodb+srv://...

# JWT
JWT_SECRET=your-secret-key
JWT_EXPIRE=7d
JWT_REFRESH_SECRET=your-refresh-secret
JWT_REFRESH_EXPIRE=30d

# البريد الإلكتروني
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_USER=your-email
EMAIL_PASS=your-password

# التطبيق
PORT=5000
NODE_ENV=development
FRONTEND_URL=http://localhost:3000
```

## 🔐 الصلاحيات

النظام يستخدم نظام صلاحيات دقيق للتحكم في وصول المستخدمين للصفحات والوظائف:

| الصلاحية | الوظائف المتاحة |
|----------|----------------|
| `dashboard` | لوحة التحكم |
| `playstation` | إدارة جلسات البلايستيشن |
| `computer` | إدارة جلسات الكمبيوتر |
| `cafe` | إدارة طلبات الكافيه |
| `menu` | إدارة قائمة الطعام |
| `billing` | الفوترة والدفع |
| `reports` | التقارير والإحصائيات |
| `inventory` | إدارة المخزون |
| `costs` | إدارة التكاليف |
| `users` | إدارة المستخدمين |
| `warehouse` | المستودع |
| `kitchenDisplay` | شاشة المطبخ |
| `soldItems` | الأصناف المباعة |

عند تسجيل حساب جديد، تُمنح الصلاحيات تلقائيًا حسب الدور:
- **admin**: جميع الصلاحيات
- **staff**: playstation + computer + cafe + menu + billing + reports + inventory + costs
- **cashier**: billing + cafe
- **kitchen**: cafe
