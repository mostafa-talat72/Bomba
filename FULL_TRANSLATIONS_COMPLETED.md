## 🎯 كيفية الاستخدام

### الاستخدام الفوري (الآن)

النظام جاهز للاستخدام فورًا:

```bash
# تشغيل المشروع
npm run dev

# افتح المتصفح
# اذهب إلى الإعدادات
# اختر أي لغة من 146 لغة
# ستعمل الواجهة بالإنجليزية (احتياطي)
```

### الترجمة اليدوية (لاحقًا)

لترجمة أي لغة:

1. **افتح ملف اللغة:**
```bash
# مثال: الإسبانية
code src/i18n/locales/es.json
```

2. **ترجم القيم (ليس المفاتيح):**
```json
{
  "common": {
    "save": "Guardar",      // ترجم هنا
    "cancel": "Cancelar",   // ترجم هنا
    "delete": "Eliminar"    // ترجم هنا
  }
}
```

3. **حدّث البيانات الوصفية:**
```json
{
  "_meta": {
    "translationStatus": "full"  // غيّر من english-template إلى full
  }
}
```

### الترجمة الآلية (خيار)

يمكنك استخدام خدمات الترجمة:

```bash
# Google Translate API (مجاني محدود)
npm install @vitalets/google-translate-api
node src/i18n/translate-with-google.js

# أو استخدم خدمات احترافية:
# - Google Cloud Translation API
# - DeepL API
# - Microsoft Translator
```

## 🎨 مثال على ملف ترجمة

```json
{
  "common": {
    "save": "Save",
    "cancel": "Cancel",
    "delete": "Delete",
    "edit": "Edit",
    "add": "Add",
    "language": "Español"  // اسم اللغة بلغتها الأصلية
  },
  "nav": {
    "dashboard": "Dashboard",
    "menu": "Menu",
    "billing": "Billing"
  },
  "_meta": {
    "languageCode": "es",
    "languageName": "Español",
    "translationStatus": "english-template",
    "fallbackLanguage": "en",
    "note": "This file contains English text as template. Translate the values to Español.",
    "createdAt": "2024-XX-XX"
  }
}
```
