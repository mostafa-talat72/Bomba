# إصلاح سريع - الطاولات لا تظهر محجوزة

## المشكلة
الطاولات محجوزة في قاعدة البيانات لكن تظهر فارغة في الشاشة.

## السبب
الفواتير تحتوي على طلبات لكن غير مرتبطة بطاولات.

## الحل (3 خطوات فقط)

### 1️⃣ افحص المشكلة
```bash
node server/scripts/checkOrderBillLinking.js
```

### 2️⃣ أصلح المشكلة
```bash
node server/scripts/fixOrderBillLinking.js
```

### 3️⃣ حدّث الصفحة
- اضغط F5 في صفحة الكافيه
- اضغط F5 في صفحة الفواتير

## ✅ النتيجة
- الطاولات المحجوزة تظهر باللون الأحمر
- جميع الفواتير تظهر في صفحة الفواتير
- الطلبات تظهر عند الضغط على الطاولة
