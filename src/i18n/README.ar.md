# نظام الترجمة والتعدد اللغوي (i18n)

## نظرة عامة
يدعم هذا التطبيق أكثر من 149 لغة عالمية مع احتياطي تلقائي للإنجليزية للغات التي لا تحتوي على ترجمات كاملة.

## اللغات المدعومة
جميع اللغات من `shared/languages.js` مدعومة، بما في ذلك:
- **لغات RTL (من اليمين لليسار)**: العربية، الأردو، العبرية، الفارسية، البشتو، السندية، الأويغورية، الديفيهي، الكشميرية
- **اللغات الرئيسية**: الإنجليزية، الفرنسية، الإسبانية، الألمانية، الإيطالية، البرتغالية، الروسية، الصينية، اليابانية، الكورية، الهندية، وغيرها الكثير
- **اللغات الإقليمية**: أكثر من 140 لغة إضافية من جميع القارات

## ملفات الترجمة الحالية
- `ar.json` - العربية - ترجمة كاملة
- `en.json` - الإنجليزية - ترجمة كاملة
- `fr.json` - الفرنسية - ترجمة كاملة

## إضافة ترجمات جديدة

### الخيار 1: إنشاء ملف ترجمة جديد
1. انسخ `en.json` كقالب
2. أنشئ ملف جديد: `locales/[رمز-اللغة].json`
3. ترجم جميع المفاتيح إلى اللغة المستهدفة
4. استورد الملف في `config.ts`:
   ```typescript
   import es from './locales/es.json';
   ```
5. أضفه إلى الموارد:
   ```typescript
   const resources: Record<string, { translation: any }> = {
     ar: { translation: ar },
     en: { translation: en },
     fr: { translation: fr },
     es: { translation: es }, // أضف لغتك الجديدة
   };
   ```

### الخيار 2: استخدام الاحتياطي التلقائي
جميع اللغات في `WORLD_LANGUAGES` تستخدم تلقائياً الترجمات الإنجليزية كاحتياطي. لا حاجة لتكوين إضافي!

### الخيار 3: استخدام أداة إنشاء القالب
```bash
node src/i18n/create-translation-template.js [رمز-اللغة]
```
مثال:
```bash
node src/i18n/create-translation-template.js es
```

## الاستخدام في المكونات
```typescript
import { useTranslation } from 'react-i18next';

function MyComponent() {
  const { t } = useTranslation();
  
  return <button>{t('common.save')}</button>;
}
```

## دعم RTL
اللغات التي تحتوي على `rtl: true` في `shared/languages.js` تقوم تلقائياً بـ:
- تعيين `dir="rtl"` على عناصر HTML
- تطبيق أنماط خاصة بـ RTL
- ضبط اتجاه التخطيط

## اختيار اللغة
يمكن للمستخدمين اختيار لغتهم المفضلة من صفحة الإعدادات، والتي تعرض جميع الـ 149 لغة بأسمائها الأصلية.

## استراتيجية الاحتياطي
1. محاولة تحميل ترجمة اللغة المحددة
2. إذا كان المفتاح مفقوداً، الرجوع إلى الإنجليزية
3. إذا كانت الإنجليزية مفقودة، عرض المفتاح نفسه

## أفضل الممارسات
1. استخدم دائماً مفاتيح الترجمة، لا تكتب النص مباشرة
2. استخدم أسماء مفاتيح وصفية: `settings.profile.title` وليس `sp1`
3. حافظ على اتساق الترجمات عبر اللغات
4. اختبر لغات RTL لمشاكل التخطيط
5. استخدم الاستيفاء للقيم الديناميكية: `t('welcome', { name: 'أحمد' })`

## المساهمة في الترجمات
للمساهمة بترجمات للغة جديدة:
1. أنشئ ملف JSON جديد في `locales/`
2. ترجم جميع المفاتيح من `en.json`
3. اختبر الترجمة في التطبيق
4. قدم طلب سحب (Pull Request)
