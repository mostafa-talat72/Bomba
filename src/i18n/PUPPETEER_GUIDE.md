# دليل استخدام Web Automation Translation

## ما هو؟

سكريبت يفتح متصفح Chrome تلقائياً ويستخدم موقع Google Translate لترجمة النصوص!

## المميزات

✅ **لا يحتاج API key** - يستخدم موقع Google مباشرة
✅ **مجاني 100%** - لا توجد حدود
✅ **ترى الترجمة بعينك** - المتصفح يفتح أمامك
✅ **جودة ممتازة** - نفس جودة Google Translate
✅ **يحفظ التقدم** - يمكن إيقافه وإعادة تشغيله

## كيف يعمل؟

1. يفتح متصفح Chrome تلقائياً
2. يذهب لموقع Google Translate
3. يكتب النص ويختار اللغة
4. ينتظر الترجمة
5. يحفظ النتيجة
6. يكرر العملية لكل نص

## التشغيل

### الخطوة 1: تأكد من تثبيت Puppeteer
```bash
npm install puppeteer --save-dev
```

### الخطوة 2: شغل السكريبت
```bash
node src/i18n/translate-puppeteer.js
```

### الخطوة 3: شاهد السحر! ✨
- سيفتح متصفح Chrome تلقائياً
- سترى الترجمة تحدث أمام عينيك
- لا تغلق المتصفح - اتركه يعمل

## الإعدادات

في ملف `translate-puppeteer.js`:

```javascript
const DELAY_BETWEEN_STRINGS = 3000; // 3 ثوانٍ بين كل نص
const DELAY_BETWEEN_LANGUAGES = 10000; // 10 ثوانٍ بين كل لغة
```

**يمكنك تعديلها:**
- زيادة التأخير = أبطأ لكن أكثر أماناً
- تقليل التأخير = أسرع لكن قد يحظرك Google

## الوقت المتوقع

- **كل نص**: ~3 ثوانٍ
- **كل لغة**: ~2.7 ساعة (3272 نص × 3 ثوانٍ)
- **67 لغة**: ~180 ساعة (7.5 يوم)

**نصيحة:** شغله واتركه يعمل طوال الليل! 🌙

## المميزات الإضافية

### 1. يحفظ التقدم تلقائياً
إذا أوقفت السكريبت، سيبدأ من حيث توقف عند إعادة التشغيل.

### 2. يظهر النص الحالي
سترى النص الذي يتم ترجمته في الوقت الفعلي.

### 3. يتعامل مع الأخطاء
إذا فشلت ترجمة، يحتفظ بالنص الأصلي ويستمر.

## استكشاف الأخطاء

### المتصفح لا يفتح
```bash
# جرب تثبيت Puppeteer مرة أخرى
npm install puppeteer --save-dev --force
```

### "Timeout waiting for selector"
- زد التأخير بين النصوص
- تأكد من اتصال الإنترنت

### Google يحظر الطلبات
- زد `DELAY_BETWEEN_STRINGS` إلى 5000 أو أكثر
- أوقف السكريبت لمدة ساعة ثم أعد المحاولة

## نصائح للأداء الأفضل

1. **اتركه يعمل ليلاً** - لن يزعجك
2. **لا تستخدم الكمبيوتر** - قد يتداخل مع المتصفح
3. **اتصال إنترنت مستقر** - مهم جداً
4. **لا تغلق المتصفح** - اتركه مفتوحاً

## المقارنة مع الطرق الأخرى

| الطريقة | السرعة | الجودة | التكلفة | السهولة |
|---------|---------|--------|---------|----------|
| Puppeteer | متوسطة | ممتازة | مجاني | سهل |
| Azure API | سريعة جداً | ممتازة | مجاني* | متوسط |
| Google API | بطيئة | ممتازة | مجاني | سهل |
| LibreTranslate | سريعة | جيدة | مجاني | سهل |

*Azure: 2 مليون حرف/شهر مجاناً

## الخلاصة

✅ **استخدم Puppeteer إذا:**
- لا تريد التسجيل في أي موقع
- لا تمانع الانتظار
- تريد رؤية الترجمة بعينك

⚠️ **استخدم Azure إذا:**
- تريد سرعة عالية
- لا تمانع التسجيل
- تريد إنهاء الترجمة في ساعات
